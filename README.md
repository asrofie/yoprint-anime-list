## Overview
Anime Search App - a two-page application where users can search for anime and view details.
- Run `npm install`
- Run `npm run dev`
- Local url: http://localhost:4000/
- Live url: https://yoprint-anime-list.vercel.app/

## Bonus Implementation

- If you provide the Anthropic API Key, it will translate the anime's synopsis on detail page

- Mobile responsiveness
